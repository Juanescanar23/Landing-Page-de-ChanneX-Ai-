# ChanneX AI Landing (Next.js)

Landing page oficial de ChanneX AI construida con Next.js 14, React y Tailwind CSS.

## Stack

- Next.js 14 (App Router)
- React 18
- Tailwind CSS
- Framer Motion

## Ejecutar en local

```bash
npm install
npm run dev
```

Abrir en: `http://localhost:3000`

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run start
```

## Estructura principal

- `app/` layout y página principal
- `components/` secciones de la landing
- `lib/landing-data.ts` contenido estructurado
- `public/` assets estáticos

## Despliegue

Generar build de producción:

```bash
npm run build
npm run start
```
